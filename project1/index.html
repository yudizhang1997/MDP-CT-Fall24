Kludging as a Creative Approach to Technology

yudi zhang | p5js | project 1 | fall 2024

Introduction

I am trying to do some Interactive Visual MIDI Experience.

Inspiration: Pianocktail

I want to explores the creative potential of interactive technology by utilizing the Web MIDI API and the p5.js library. Inspired by dynamic audio-visual experiences, the goal was to map MIDI keyboard inputs to colorful 3D visualizations. These interactions create an engaging and immersive experience where sound transforms into vibrant motion.

Work Process

Core Parts

  • MIDI Keyboard - YAMAHA P255: My equipment and it acts as the input device to trigger visual effects.
  • Web MIDI API: Facilitates communication between the MIDI keyboard and the browser.
  • p5.js Visualization: Maps MIDI input to 3D animated bars that respond dynamically.

Process Documentation

Started with p5.js and basic MIDI input setup. The initial prototype visualized simple bar animations triggered by MIDI keypresses.

Adding bar rotations, height transitions, and color changes based on velocity inputs from the MIDI device.
