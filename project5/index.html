1st Shader Practice: Plaid Pattern

Core Concept

The plaid pattern is a combination of horizontal and vertical stripes that can be superimposed using a mathematical formula.

Practices & Code

Use sin or step functions to generate stripes in the horizontal and vertical directions and combine them into a grid.
