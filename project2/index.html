Work Process

Core Parts

  • MIDI Keyboard - YAMAHA P255: My equipment and it acts as the input device to trigger visual effects.
  • Web MIDI API: Facilitates communication between the MIDI keyboard and the browser.
  • p5.js Visualization: Maps MIDI input to 3D animated bars that respond dynamically.

Process Documentation

Started with p5.js and basic MIDI input setup. The initial prototype visualized simple bar animations triggered by MIDI keypresses.

Adding bar rotations, height transitions, and color changes based on velocity inputs from the MIDI device.
